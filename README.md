# ZMConfuse
ZMConfuse 是一个命令行脚本。用于对使用Objective-C为开发语言的应用进行代码混淆。混淆的内容包括文件名、类名、协议名、函数名。
混淆实际是对上述内容的关键字提取之后进行md5加密，使得工程的可读性降低，已达到增加应用安全性的目的。
